# הוראות התקנה - מזג אוויר API

## שלב 1: השגת API Key מ-OpenWeatherMap

1. **היכנס לאתר OpenWeatherMap:**
   - פתח את הדפדפן ולך לכתובת: https://openweathermap.org/api

2. **הירשם/התחבר:**
   - לחץ על "Sign Up" אם אין לך חשבון
   - או "Sign In" אם כבר יש לך חשבון
   - מלא את הפרטים הנדרשים (שם משתמש, אימייל, סיסמה)

3. **קבלת API Key:**
   - לאחר ההתחברות, לחץ על "API keys" בתפריט החשבון
   - תראה את ה-API key שלך (או תוכל ליצור אחד חדש)
   - העתק את ה-API key (זה מחרוזת ארוכה של תווים)

## שלב 2: הגדרת API Key בפרויקט

1. **צור קובץ .env בתיקיית client:**
   - בתיקיית `client`, צור קובץ חדש בשם `.env`
   - **חשוב:** הקובץ צריך להיות בדיוק בשם `.env` (לא `.env.txt` או משהו אחר)

2. **הוסף את ה-API key לקובץ:**
   - פתח את הקובץ `.env` בעורך הטקסט
   - הוסף את השורה הבאה:
   ```
   VITE_WEATHER_API_KEY=הדבק_כאן_את_ה_API_key_שלך
   ```
   - החלף את `הדבק_כאן_את_ה_API_key_שלך` ב-API key שהעתקת

   **דוגמה:**
   ```
   VITE_WEATHER_API_KEY=1234567890abcdef1234567890abcdef
   ```

3. **שמור את הקובץ**

## שלב 3: הפעלה מחדש של השרת

- **חשוב:** לאחר יצירת/עדכון קובץ `.env`, צריך להפעיל מחדש את שרת הפיתוח:
  - עצור את השרת (Ctrl+C)
  - הפעל מחדש עם `npm run dev`

## שלב 4: בדיקה

1. פתח את האפליקציה בדפדפן
2. לך לדף הבית
3. הדפדפן יבקש ממך אישור לגישה למיקום שלך - לחץ "Allow"
4. אמור להופיע וידג'ט מזג האוויר עם המידע הנוכחי

## פתרון בעיות

### הבעיה: "API key לא הוגדר"
- **פתרון:** ודא שיצרת קובץ `.env` בתיקיית `client` ושהשם שלו בדיוק `.env`
- ודא שהוספת את השורה `VITE_WEATHER_API_KEY=...` עם ה-API key שלך
- ודא שהפעלת מחדש את השרת לאחר יצירת הקובץ

### הבעיה: "לא ניתן לקבל את המיקום שלך"
- **פתרון:** ודא שהדפדפן מאפשר גישה למיקום
- בדפדפן Chrome/Edge: לחץ על האייקון של המנעול בשורת הכתובת > Site settings > Location > Allow
- ודא שהאתר רץ על HTTPS (או localhost)

### הבעיה: "שגיאה בטעינת נתוני מזג האוויר"
- **פתרון:** ודא שה-API key תקין ופעיל
- בדוק ב-OpenWeatherMap שהחשבון שלך פעיל
- התוכנית החינמית מאפשרת 60 קריאות לדקה - אם חרגת, תצטרך לחכות

### הבעיה: מזג האוויר לא מופיע
- **פתרון:** פתח את Console בדפדפן (F12) ובדוק אם יש שגיאות
- ודא שהקומפוננטה `WeatherWidget` מיובאת נכון ב-`Home.tsx`

## מידע נוסף

- **תוכנית חינמית:** 60 קריאות לדקה, 1,000,000 קריאות לחודש
- **תמיכה:** אם יש בעיות, בדוק את התיעוד: https://openweathermap.org/api
- **אבטחה:** לעולם אל תעלה את קובץ `.env` ל-GitHub! הוא כבר מופיע ב-.gitignore

## הערות חשובות

1. הקובץ `.env` לא נשמר ב-Git (הוא ב-.gitignore) - זה טוב לאבטחה
2. כל מפתח פיתוח צריך ליצור קובץ `.env` משלו עם ה-API key שלו
3. בייצור (production), תצטרך להגדיר את המשתנה בסביבת הייצור (Heroku, Vercel וכו')
